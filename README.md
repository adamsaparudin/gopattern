# GoPattern

Go pattern with Golang Native using PostgreSQL, JWT & GORM.


## Env Config
Change *.env.example* to *.env*

| KEY                       | Value | 
| -----------------------   | ------ 
| DB_HOST                   | 127.0.0.1
| DB_PORT                   | 5432
| DB_USER                   | postgres
| DB_NAME                   | gopattern    
| DB_PASSWORD               | yourdbpassword
| SECRET                    | secretJWT
